Section 02

Onboarding — 2 phases

Phase 1 ship trước (không cần UA). Phase 2 thêm creative tag khi UA team sẵn sàng. Cả 2 dùng chung luồng 2-pack offer.

Phase 1 — Geo + climate + DNA (ship trước)
Step 1: Store location → country → Zone Season Map → chọn first outfit theo culture + climate.
VD: VN Tết = áo dài, Indonesia tháng 7 = monsoon look, US December = holiday party.

Step 2: Style DNA Quiz — 5 câu A/B chọn outfit → output Style DNA (VD: Elegant 60%, Street 30%).

Step 3: Offer 2 IAP packs:
    Pack A = based on Style DNA (VD: DNA = Elegant → "Elegant Starter Pack")
    Pack B = based on culture/season (VD: VN Tết → "Lunar New Year Pack")

Step 4: Giới thiệu Hall of Fame — "Login 7 ngày liên tiếp = nhận 1 bộ sưu tập!"
Phase 2 — thêm UA creative tag (khi UA team sẵn sàng)
Step 1: User thấy UA creative (VD: ad về "Fantasy Night") → install → game bắt creative tag từ Adjust/AppsFlyer.

Step 2: Creative tag → chỉ định topic đầu tiên users được chơi = match creative đã xem. User trải nghiệm đúng thứ đã hấp dẫn họ.

Step 3: Style DNA Quiz (vẫn 5 câu).

Step 4: Offer 2 IAP packs — logic thông minh:
    Pack A = based on DNA result
    Pack B = based on creative tag topic
    Nếu DNA trùng creative tag → thay Pack B = culture/season pack hoặc complementary style.

Step 5: Giới thiệu Hall of Fame.
2-pack logic: Luồng 2 packs sẵn sàng từ Phase 1. Phase 2 chỉ thêm tín hiệu creative tag → thay đổi Pack B source. Code framework giống nhau.

Section 04

10-day event — 3 fixed sub-themes

Event = 1 Major Category (VD: Vacation). Chọn 3 sub-themes từ đó. D1-D3 intro lần lượt. D4-D10 cả 3 open.

3 sub-themes (example: event "Vacation")
Sub-themeIntro daySetsDifficulty
Summer BeachD11 craft/coin + 1 IAPEasy — free 3★ đủ Good
Mountain RetreatD21 craft/coin + 1 IAPMedium — 4-5★ help
Cruise NightD31 craft/coin + 1 IAPHard — 6★ for Perfect
D4-D10: all 3 open — users tự do xoay vòng, thu thập, cải thiện
Daily player flow
1. Browse: Xem items cho sub-theme. Auto-recommend 3 combos (3 free/ngày). Ready sets = tap auto-dress.

2. Collect: Free drop, event currency, coins, craft, IAP, ad try-on.

3. Style: Phối outfit. Customization upsell (đổi màu/hiệu ứng/phụ kiện $$).

4. Submit: Submit → BE chấm → tier (Pass/Good/Great/Perfect) = base score (A).

5. Vote pool: Outfit vào feed → users cho tim (1/2/3 ♥) → vote bonus cộng vào score.

6. Ranking: Base + votes + love ratio + hour bonus = day total. Tổng 10 ngày x 3 themes = final rank.
Catch-up + Hall of Fame
Catch-up: Bỏ lỡ ngày trước → làm bù nhưng score x0.7. Đủ để không bỏ cuộc.

Hall of Fame — login 7 ngày = 1 collection set:
Mỗi ngày login nhận 1 item thật (dùng được ngay, không ghép mảnh):

Ngày 1: Dress (3★)
Ngày 2: Shoes (3★)
Ngày 3: Accessory (3★)
Ngày 4: Hair (3★)
Ngày 5: Bag (3★)
Ngày 6: Hat (3★)
Ngày 7: Bonus Item (4★)

Đủ 7 ngày = bộ hoàn chỉnh + set harmony bonus. Bỏ lỡ → mua item thiếu bằng event currency hoặc ads.
Section 05

Scoring — 5 sources, 1 formula

Scale x100 để điểm trông lớn. 4 nguồn tính ranking (A+B+C+E). 1 nguồn cho economy riêng (D).

Formula mỗi ngày per theme:

(A base score + B vote hearts) = subtotal
× C love ratio bonus % = adjusted total
+ E rank hour bonus = Day total

+ D quest rewards → event currency (tách riêng)

Day total x 3 themes x 10 ngày = Event total → leaderboard.
A — Base score (BE chấm)
Item stars: 3★ = 3,000 / 4★ = 4,000 / 5★ = 5,000 / 6★ = 6,000
Topic match: main +800 / sub +400 / no match +0
Style fit: SSS +600 / SS +480 / S +360 / A +240 / B +120
Unworn: 1st +300 / 2nd +150 / 3rd+ = 0
Set harmony: +200-500 (items cùng collection)
Customization: +100-300 (color/effect/accessory)

→ Tier: Pass 1,000-3,900 / Good 4,000-6,400 / Great 6,500-8,400 / Perfect 8,500-10,000
B — Vote hearts
♥ x1 = 100 pts per vote
♥♥ x2 = 200 pts per vote
♥♥♥ x3 = 300 pts per vote

Tổng = sum votes x giá trị. Realtime update.
C — Love ratio bonus
% voters cho 3 ♥♥♥ → bonus % lên subtotal (A+B):

30-49% voters cho 3 hearts → bonus +5%
50-69% → bonus +10%
70-89% → bonus +15%
90%+ → bonus +20%
D — Quest rewards (= event currency, KHÔNG ranking)
Submit 1 outfit → +200 currency
Vote 10 outfits → +300 currency
Try auto-recommend 3x → +100 currency
Collect 2 items → +200 currency
Login (Hall of Fame) → +1 item

Quests = economy. KHÔNG cộng ranking. 2 hệ tách biệt.
E — Rank hour bonus (passive, tự động)
% trên base score (A), tích lũy mỗi giờ. Giữ rank = điểm tự tăng:

Newcomer +0.25%/h / Apprentice +0.50%/h / Stylist +0.75%/h / Designer +1.00%/h
Creative Director +1.10%/h / Style Icon +1.25%/h / Trendsetter +1.40%/h / Fashion Queen +1.75%/h

VD: Designer, base 7,500 pts, 24h → 7,500 x (1% x 24) = +1,800 pts tự động.
Scoring example: Day 5, rank Gold, outfit Great
A. Base (Great): 7,500 (stars 5,000 + topic +800 + style SS +480 + unworn +300 + harmony +400 + custom +200)
B. Votes: 12x ♥♥♥=3,600 + 6x ♥♥=1,200 + 3x ♥=300 = +5,700
Subtotal: 13,200
C. Love ratio: 12/21 = 57% → +10% = +1,320
Adjusted: 14,520
E. Hour bonus (Designer 1%/h x 24h): 7,500 x 24% = +1,800
Day 5 total: 16,320 pts

+ Quests: 4 done → +800 event currency (mua đồ)
10-day simulation: 3 themes combined
Theme A (Summer Beach): 154,620 pts
Theme B (Mountain Retreat): 124,900 pts
Theme C (Cruise Night): 108,400 pts

Event total: 387,920 pts → event leaderboard ranking.

Điểm tăng dần vì: collect thêm items, vote tích lũy, hour bonus tăng theo rank, harmony bonus.
Section 06

Heart voting + love ratio bonus

Voting bằng tim. Feed scroll. Realtime update. Love ratio = % 3-heart → bonus %.

How voting works
Voter mở tab voting:
→ Outfits hiện dạng feed scroll (giống Instagram)
→ Random từ pool cùng sub-theme
→ Tap cho tim: 1 ♥ / 2 ♥♥ / 3 ♥♥♥
→ Next outfit → repeat

Pool distribution:
→ Mỗi outfit hiện cho ~20-50 voters (cap để fair)
→ Outfits mới ưu tiên hiện trước
→ Inhouse team vote trước (seed), users vote lẫn nhau sau

Voter rewards: Mỗi 10 votes = earn event currency
Score update: Realtime — mỗi vote mới = điểm tăng ngay

Section 09

Economy + encore + lifecycle

Event currency (temporary)
Earn from: quests, voting (per 10 votes), scoring milestones, Hall of Fame.
Spend on: items 3-5★, craft materials. 6★ = IAP only.
Expiry: Reset khi event kết thúc. Spend it or lose it.
Buy packs: $0.99 (100) / $4.99 (600) / $9.99 (1500).
Item lifecycle + encore
During event: Items available in shop. Full scoring bonus.
Event ends: Items leave shop. Đã mua = sở hữu vĩnh viễn.
Encore (3-4 tuần sau): Items return cho users chưa sở hữu. Price = 120%.
Season bonus decay: Current season = 100%. Last season = 70%. 2+ = 50%.
Unworn decay: 1st use = full. 2nd = 50%. 3rd+ = 0%. Reset mỗi season.
